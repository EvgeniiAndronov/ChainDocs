# 🌐 Web UI - Категории и Массовая Загрузка

## 📁 Новые возможности Web UI

### 1. Управление категориями

**Страница:** `/web/categories`

**Возможности:**
- ✅ Создание новых категорий
- ✅ Просмотр списка всех категорий
- ✅ Просмотр количества документов в категории
- ✅ Удаление категорий
- ✅ Статистика по категориям

**Как использовать:**
1. Откройте `/web/categories`
2. Заполните форму создания категории:
   - **ID:** уникальный идентификатор (латиницей)
   - **Название:** понятное имя категории
   - **Описание:** опционально
3. Нажмите "Создать категорию"

---

### 2. Массовая загрузка документов

**Страница:** `/web/bulk-upload`

**Возможности:**
- ✅ Drag & Drop загрузка файлов
- ✅ Выбор категории для файлов
- ✅ Загрузка до 100MB за раз
- ✅ Прогресс бар загрузки
- ✅ Результаты по каждому файлу
- ✅ Статистика успешных/неуспешных загрузок

**Как использовать:**
1. Откройте `/web/bulk-upload`
2. Выберите категорию (опционально)
3. Перетащите файлы в область загрузки или нажмите "Выбрать файлы"
4. Нажмите "Загрузить N файлов"
5. Дождитесь завершения и просмотрите результаты

## 🔧 API Endpoints

### Категории

| Endpoint | Метод | Описание |
|----------|-------|----------|
| `/api/categories` | GET | Список категорий |
| `/api/categories` | POST | Создать категорию |
| `/api/categories/{id}` | GET | Информация о категории |
| `/api/categories/{id}/documents` | GET | Документы категории |
| `/api/categories/{id}` | DELETE | Удалить категорию |

### Загрузка

| Endpoint | Метод | Описание |
|----------|-------|----------|
| `/api/upload` | POST | Загрузка одного файла |
| `/api/upload/bulk` | POST | Массовая загрузка |

## 💡 Примеры использования

### Создание категории через API

```bash
curl -X POST http://localhost:8080/api/categories \
  -H "Content-Type: application/json" \
  -d '{
    "id": "diplomas_2026",
    "name": "Дипломы 2026",
    "description": "Дипломы выпускников 2026 года"
  }'
```

### Массовая загрузка через API

```bash
curl -X POST http://localhost:8080/api/upload/bulk \
  -F "category=diplomas_2026" \
  -F "files=@diploma1.pdf" \
  -F "files=@diploma2.pdf" \
  -F "files=@diploma3.pdf" | jq
```

## ⚙️ Технические детали

### Frontend
- **Alpine.js** - реактивность
- **Bootstrap 5** - стилизация
- **Drag & Drop API** - загрузка файлов
- **XMLHttpRequest** - прогресс загрузки

### Backend
- **Go** - обработка запросов
- **BBolt** - хранение метаданных категорий
- **SHA-256** - хэширование файлов

### Ограничения
- Макс. размер: **100 MB** за запрос
- Рекомендуется: **20-50 файлов** за раз
- Тип файлов: **Только PDF**

## 🎯 Best Practices

### Для категорий
1. Используйте понятные ID: `diplomas_bakalavry`, `dogovory_2026`
2. Группируйте по типу и году: `diplomas_2026`, `diplomas_2027`
3. Не создавайте слишком много: 10-20 категорий достаточно

### Для массовой загрузки
1. Разбивайте на пакеты по 20-50 файлов
2. Проверяйте размер перед загрузкой
3. Используйте категории для организации
4. Сохраняйте результаты загрузки
