## 🎯 План реализации веб-интерфейса

### Технологии
- **Frontend:** HTMX + Alpine.js (лёгкий, без сборки)
- **Backend:** Встроен в сервер (Go template)
- **Стили:** Bootstrap 5 + Material Design иконки

### Структура страниц

```
web/
├── templates/
│   ├── base.html         # Базовый шаблон
│   ├── index.html        # Dashboard
│   ├── blocks.html       # Список блоков
│   ├── block.html        # Детали блока
│   ├── upload.html       # Загрузка
│   └── keys.html         # Ключи
├── static/
│   ├── css/
│   │   └── style.css
│   └── js/
│       └── app.js
└── handlers.go           # HTTP handlers для web
```

### API для веба

```
GET  /web/              → Dashboard
GET  /web/blocks        → Список блоков
GET  /web/blocks/{hash} → Детали блока
GET  /web/upload        → Форма загрузки
POST /web/upload        → Загрузка файла
GET  /web/keys          → Список ключей
GET  /web/consensus     → Мониторинг консенсуса
```
